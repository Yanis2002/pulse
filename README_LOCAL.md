# Локальная версия PULSE | CLUB

## Запуск локальной версии

Локальная версия использует отдельную базу данных и предназначена для разработки и тестирования через туннелирование.

### Быстрый старт:

```bash
./local/start_local.sh
```

Или вручную:

```bash
python3 local/app_local.py
```

### Настройка туннелирования:

1. **Запустите локальный сервер:**
   ```bash
   ./local/start_local.sh
   ```

2. **Настройте туннель** (например, через tunnel4.com или ngrok):
   - Локальный адрес: `http://localhost:8000`
   - Публичный адрес будет предоставлен сервисом туннелирования

3. **Настройте webhook для бота:**
   - Используйте публичный адрес туннеля
   - URL webhook: `https://ваш-туннель-адрес/api/telegram/webhook`

### Отличия локальной версии:

- ✅ Отдельная база данных в папке `local_db/`
- ✅ Режим отладки включен
- ✅ Не влияет на продакшн базу данных
- ✅ Можно тестировать без деплоя

### Структура:

```
poker/
├── app.py                  # Основное приложение (продакшн)
├── pulse_tournaments.db    # Продакшн база данных
├── local/                  # Локальная версия (не деплоится)
│   ├── app_local.py        # Локальная версия приложения
│   └── start_local.sh      # Скрипт запуска локальной версии
└── local_db/               # Локальная база данных
    └── pulse_tournaments.db
```

### Важно:

- Локальная версия НЕ синхронизируется с продакшн версией
- Данные в локальной версии хранятся отдельно
- Для тестирования бота используйте туннель с HTTPS
